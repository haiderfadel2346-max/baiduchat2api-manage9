### How to Take Screenshots:

```bash
# 1. Start the application
npm run dev

# 2. Login with admin / sk-admin

# 3. Navigate to each page and take screenshots

# 4. Save them in this directory with the names above

# 5. Update README.md image links
```

### Image Guidelines:

- Resolution: 1920x1080 or higher
- Format: PNG (preferred) or JPG
- Size: < 500KB per image (use compression if needed)
- Show the key features clearly
- Blur any sensitive information (if any)
