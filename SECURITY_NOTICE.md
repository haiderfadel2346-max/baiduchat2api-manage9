# ⚠️ 安全提醒

## 本文档包含示例配置，请勿直接使用！

本仓库的文档中的所有密钥和配置都是**示例值**，仅用于演示目的。

### 🔐 生产环境必须修改

1. **NEXTAUTH_SECRET**: 必须重新生成
   ```bash
   openssl rand -base64 32
   ```

2. **默认密码**: 首次登录后立即修改
   - 默认: `admin` / `sk-admin`
   - 请修改为强密码

3. **数据库文件**: 已在 `.gitignore` 中排除

### ✅ 安全检查清单

在生产环境部署前：
- [ ] 重新生成 NEXTAUTH_SECRET
- [ ] 修改默认管理员密码
- [ ] 配置环境变量（不要硬编码）
- [ ] 启用 HTTPS
- [ ] 配置防火墙规则
- [ ] 定期备份数据库

### 🚫 不要提交的文件

这些文件已在 `.gitignore` 中：
- `.env.local`
- `data/*.db`
- `*.db-shm`
- `*.db-wal`
